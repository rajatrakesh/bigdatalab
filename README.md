## Cloudera Big Data Lab

#### Pre-requisites:
* Cloudera Quick Start VM (Kudu Not installed)
* Working Internet Connection
* Oracle VM Virtualbox (preferred) or Vmware
* Basic working knowledge of CDH and SQL
* Laptop with minimum 16 GB RAM
